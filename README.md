## 🔥 나만의 Best Practice 원칙
1. **가독성 우선**: 변수명은 `a, b, c` 대신 `node_count`, `is_visited`처럼 직관적으로.
2. **효율성 체크**: 시간 복잡도($O(N)$)를 항상 계산하고 제출하기.
3. **기록의 힘**: 왜 이 알고리즘을 선택했는지 주석으로 '생각의 흐름' 남기기.
